# Figure label note

The original BHSA lexeme codes are preserved in parentheses for scholarly traceability.
English glosses are added to make the figures more readable for non-specialist viewers.

Example:
- `say (>MR[)` means the English gloss is "say" and the BHSA lexeme code is `>MR[`.
- `offering (QRBN/)` means the English gloss is "offering" and the BHSA lexeme code is `QRBN/`.

The friendly figures are intended for posters, presentations, and GitHub viewing.
The original CSV tables preserve the full underlying lexeme data.
